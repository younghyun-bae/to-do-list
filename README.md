# 📋 To Do List

- One of the toyprojects made with React.js / post CSS

## 🔍 Project description

Users can

- add a to-do item ⌨️
- toggle an item as either active or completed ✅
- see filtered items by status 🗂
- remove each item 🗑

## 💻 Set-up

to install dependencies

`npm install`

torun the application

`npm start`
